# DMN Fitness - Gimnasio TFG

**Proyecto final de grado (TFG)**: Sistema web para la gestión de un gimnasio, desarrollado con **PHP (CodeIgniter 4)**.

Realizado por Narcis Claudiu Aducovschi , Matias Bolagay Quevedo y Danny Sánchez

---

## 📌 Descripción

DMN Fitness es una aplicación web que permite gestionar usuarios, pagos, y acceso a servicios de un gimnasio.  
Incluye funcionalidades de **registro, login, gestión de pagos y control de sesiones**.  

Este proyecto está pensado para ser un ejemplo de **aplicación MVC** con buenas prácticas en PHP.
